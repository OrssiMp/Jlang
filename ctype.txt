Voici une liste complète des fonctions PHP de l'extension `ctype` qui vérifient le type de caractères dans une chaîne. Chaque fonction retourne `true` si tous les caractères de la chaîne respectent le critère spécifié, et `false` sinon.

### Fonctions ctype de base :

1. **`ctype_alnum`** - Vérifie si tous les caractères sont alphanumériques (a-z, A-Z, 0-9)
   ```php
   ctype_alnum("abc123")  // true
   ctype_alnum("abc#123") // false
   ```

2. **`ctype_alpha`** - Vérifie si tous les caractères sont des lettres (a-z, A-Z)
   ```php
   ctype_alpha("abcXYZ")  // true
   ctype_alpha("abc123")  // false
   ```

3. **`ctype_cntrl`** - Vérifie les caractères de contrôle (comme \n, \r, \t)
   ```php
   ctype_cntrl("\n\r\t")  // true
   ctype_cntrl("abc")     // false
   ```

4. **`ctype_digit`** - Vérifie les chiffres uniquement (0-9)
   ```php
   ctype_digit("12345")   // true
   ctype_digit("12.34")   // false
   ```

5. **`ctype_graph`** - Vérifie les caractères imprimables sauf l'espace
   ```php
   ctype_graph("abc123")  // true
   ctype_graph("abc 123") // false (à cause de l'espace)
   ```

6. **`ctype_lower`** - Vérifie les lettres minuscules uniquement
   ```php
   ctype_lower("abc")     // true
   ctype_lower("Abc")     // false
   ```

7. **`ctype_print`** - Vérifie les caractères imprimables (y compris l'espace)
   ```php
   ctype_print("abc 123") // true
   ctype_print("abc\n123")// false (à cause de \n)
   ```

8. **`ctype_punct`** - Vérifie les caractères de ponctuation
   ```php
   ctype_punct("!@#")     // true
   ctype_punct("a!@#")    // false
   ```

9. **`ctype_space`** - Vérifie les caractères d'espacement
   ```php
   ctype_space(" \n\r\t") // true
   ctype_space(" a ")     // false
   ```

10. **`ctype_upper`** - Vérifie les lettres majuscules uniquement
    ```php
    ctype_upper("ABC")     // true
    ctype_upper("AbC")     // false
    ```

11. **`ctype_xdigit`** - Vérifie les caractères hexadécimaux (0-9, a-f, A-F)
    ```php
    ctype_xdigit("1aF3")   // true
    ctype_xdigit("1g")     // false
    ```

### Points importants :
- Toutes ces fonctions retournent `false` si la chaîne est vide
- Les chaînes non-vides sont converties en entiers si possible avant vérification
- Les fonctions sont sensibles à la casse (sauf indication contraire)
- Ces fonctions sont généralement plus rapides que les expressions régulières pour des vérifications simples
